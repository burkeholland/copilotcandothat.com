Submit a Tip

Want to share a GitHub Copilot tip? Follow these steps to submit your tip:

1. Create the Video

  • Record a short, focused video demonstrating your Copilot tip
  • Upload it to YouTube
  • Make sure the video clearly shows Copilot in action

2. Create a Pull Request

  • Fork the repository
  • Create a new branch for your tip
  • Add a new markdown file in the _tips directory

3. Format Your Tip

Use this template for your markdown file:

---
title: Your Tip Title
description: A brief description of what your tip demonstrates
videoId: YOUR_YOUTUBE_VIDEO_ID
layout: tip
tags: [relevant, tags, here]
---

4. Submit Your PR

  • Ensure your tip follows the existing format
  • Submit a pull request to the main repository
  • Wait for review and approval

Tips should be focused on GitHub Copilot features and capabilities. Make sure your demonstration is clear and provides value to other developers.
